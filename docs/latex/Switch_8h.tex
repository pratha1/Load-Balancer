\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classSwitch}{Switch}} class. 

The \mbox{\hyperlink{classSwitch}{Switch}} simulates a network router that distributes incoming requests to multiple Load\+Balancers based on job type.

Routing Policy\+:
\begin{DoxyItemize}
\item \textquotesingle{}P\textquotesingle{} (Processing) → processing \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}}
\item \textquotesingle{}S\textquotesingle{} (Streaming) → streaming \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}}
\end{DoxyItemize}

This implements the bonus objective of concurrent load balancers with job separation.
