\doxysubsection{Detailed Description}
Entry point for the Load Balancer simulation. 

This program simulates a scalable web server load balancing system.

User Inputs\+:
\begin{DoxyItemize}
\item Initial number of servers
\item Simulation duration (clock cycles)
\end{DoxyItemize}

The program then\+:
\begin{DoxyEnumerate}
\item Creates a \mbox{\hyperlink{classSwitch}{Switch}} object
\item Runs the simulation
\item Outputs final statistics 
\end{DoxyEnumerate}
